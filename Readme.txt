Ceci est un "jeu de la vie" en C++/QT4/OpenGL.
Les dll de QT4.1.2 sont requisent pour ce jeu.

Le "CasesEngine" est un système de gestion de jeu comportant des cases.
Il est conçu selon la méthode UML avec l'aide du logiciel BOUML.

Les touches sont:
'Clic Gauche'    => Case vivante
'Clic droit'     => Case morte
'Clic milieu'	 => Afficher HUD
'+'              => Génération suivante
'Alt' + 'Entrée' => Plein écran
'Alt' + 'G'      => Grille on/off
'*'		 => Fin du monde
